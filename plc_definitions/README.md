# PLC Definitions Directory

此目录存放 PLC 数据块定义和变量表文件，作为项目参考文档使用。

## 文件说明

| 文件 | 说明 |
|------|------|
| `GLABAL.db` | 西门子 PLC 全局数据块定义文件，包含 DB1 等数据块的结构和地址映射 |
| `PLCValues.xlsx` | PLC 变量表 Excel 文件，包含所有变量的名称、地址、数据类型等信息 |

## 用途

- **GLABAL.db**: 用于参考 PLC 程序中的数据块结构，确保数据采集器正确解析 PLC 数据
- **PLCValues.xlsx**: 用于对照变量名和地址，确保配置文件中定义的变量与实际 PLC 程序一致

## 注意事项

这些文件仅作为参考，运行时系统会直接从 PLC 设备读取数据，不需要依赖这些文件。
