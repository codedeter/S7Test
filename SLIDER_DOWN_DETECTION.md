# 滑块下行异常检测系统

## 概述

本系统提供滑块下行异常检测功能，能够在检测到滑块下行指令发出但实际未执行时，推理出具体的前置条件不满足原因。

## 核心组件

### 1. 滑块下行异常检测器

**文件**: `src/analysis/slider_down_detector.py`

**主要类**:

| 类名 | 说明 |
|------|------|
| `ConditionStatus` | 条件状态枚举（满足/不满足/未知） |
| `SliderDownCondition` | 滑块下行前置条件定义 |
| `SliderDownAbnormalDetector` | 异常检测器主类 |

**主要方法**:

| 方法 | 说明 |
|------|------|
| `update_facts(facts)` | 更新PLC数据到检测器 |
| `check_abnormal()` | 检查是否存在异常 |
| `get_abnormal_reasoning()` | 生成详细的推理报告 |

### 2. 集成到数据处理流程

**文件**: `src/services/data_processor.py`

检测器在数据处理流程中被自动调用，无需手动干预。

## 滑块下行前置条件

基于梯形图分析，滑块下行需要满足以下10个前置条件：

| 序号 | 条件名称 | 变量名 | 期望值 | 说明 |
|------|----------|--------|--------|------|
| 1 | 急停合格 | 急停合格 | 0 | 急停按钮未按下 |
| 2 | 滑块在上限 | 滑块上限 | 1 | 滑块处于上限位置 |
| 3 | 双手合格 | 双手合格 | 1 | 双手操作按钮合格 |
| 4 | 电机启动主控 | 电机启动主控 | 1 | 电机主控已启动 |
| 5 | 允许下行 | 允许下行 | 1 | 系统允许滑块下行 |
| 6 | 移动台合格 | 移动台合格 | 1 | 移动台位置合格 |
| 7 | 驱动器正常 | 驱动器正常 | 0 | 驱动器无故障 |
| 8 | 系统无错误 | 系统Error | 0 | 系统无错误信号 |
| 9 | 安全爪打开 | 安全爪打开到位 | 1 | 安全爪已打开到位 |
| 10 | 安全爪主控 | 安全爪主控 | 1 | 安全爪主控激活 |

## 异常检测逻辑

### 检测条件

异常判定需要同时满足以下条件：

1. **有下行指令** - 检测到滑块下行/慢下/3Y1/打开充液阀信号
2. **有条件不满足** - 至少一个前置条件未满足
3. **滑块未移动** - 滑块速度为0或绝对值小于0.1
4. **超过容忍时间** - 已超过延迟容忍(1.5s) + 误差容忍(1s) = 2.5s

### 容忍时间设计

| 容忍类型 | 时间 | 说明 |
|----------|------|------|
| 延迟容忍 | 1.5秒 | 考虑信号传输延迟 |
| 误差容忍 | 1.0秒 | 考虑系统响应时间 |
| 总容忍 | 2.5秒 | 延迟+误差容忍时间 |

## 使用方法

### 基本使用

```python
from src.analysis.slider_down_detector import create_slider_detector

# 创建检测器
detector = create_slider_detector()

# 更新PLC数据
plc_data = {
    '急停合格': 0,
    '滑块上限': 1,
    '双手合格': 1,
    '电机启动主控': 1,
    '允许下行': 1,
    '移动台合格': 1,
    '驱动器正常': 0,
    '系统Error': 0,
    '安全爪打开到位': 1,
    '安全爪主控': 1,
    '滑块慢下': 1,  # 下行指令
}
detector.update_facts(plc_data)

# 检查异常
result = detector.check_abnormal()
if result['abnormal']:
    # 获取推理报告
    report = detector.get_abnormal_reasoning()
    print(report)
```

### 检查结果结构

```python
{
    'abnormal': False,              # 是否检测到异常
    'description': '',              # 异常描述
    'unsatisfied_conditions': [],   # 不满足的条件列表
    'all_conditions': [],           # 所有条件状态
    'elapsed_time': 0,              # 下行指令已激活时间
    'delay_tolerance': 1.5,         # 延迟容忍时间(秒)
    'error_tolerance': 1.0          # 误差容忍时间(秒)
}
```

## 故障规则集成

滑块下行异常检测已集成到故障规则引擎中，相关规则ID：

| 规则ID | 规则名称 | 说明 |
|--------|----------|------|
| F001 | 滑块下行异常 | 下行指令发出但滑块未移动 |
| F001_1 | 急停阻止下行 | 急停按下阻止滑块下行 |
| F001_2 | 双手不合格 | 双手操作不合格 |
| F001_3 | 电机未启动 | 电机主控未启动 |
| F001_4 | 驱动器故障 | 驱动器检测到故障 |
| F001_5 | 系统错误 | PLC检测到系统错误 |
| F001_6 | 不允许下行 | 系统未允许下行 |
| F001_7 | 移动台不合格 | 移动台位置不合格 |
| F001_8 | 安全爪未打开 | 安全爪未完全打开 |
